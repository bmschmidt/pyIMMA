How to use this dataset
=======================

This package is kept under version control in a `git repository <https://en.wikipedia.org/wiki/Git>`_. The repository is hosted on `GitHub <https://github.com/>`_ (and the documentation made with `GitHub Pages <https://pages.github.com/>`_). The repository is `<https://github.com/philip-brohan/pyIMMA>`_.

If you are familiar with GitHub, you already know `what to do <https://github.com/philip-brohan/pyIMMA>`_: If you'd prefer not to bother with that, you can download the whole package as `a zip file <https://github.com/philip-brohan/pyIMMA/archive/master.zip>`_.

To use this package, clone or download the repository and then install it with:

.. code-block:: sh

   python setup.py install --user
